Design System
A collection of reusable components, guided by clear standards, that can be assembled together to build applications.
SCSS
Syntactically Awesome Style Sheets - a CSS preprocessor that adds features like variables, nesting, and mixins.
Utility-First
A CSS approach where you compose designs by applying pre-existing classes rather than writing custom CSS.

Code Elements

Inline Code

To install the framework, run npm install tailords in your terminal. You can import it with @use "tailords" as *; in your SCSS file.

Code Blocks

// Example SCSS usage
@use "tailords" as * with (
  $theme: (
    brand: #dd224e,
    success: #77c23d,
  ),
  $enable-utility-class: true,
  $enable-css-variables: true
);

.my-component {
  background-color: color(brand);
  padding: size(double);
  border-radius: radius(medium);
}
